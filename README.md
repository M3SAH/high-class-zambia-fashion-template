# Luxury Clothing Template

A premium, elegant, mobile-first **single-page website template** designed for high-class clothing boutiques in Zambia — specializing in luxury suits, designer dresses, premium chitenge fashion, evening wear, executive outfits, and exclusive tailored pieces.

Perfect for upscale boutiques in Chingola, Kitwe, Lusaka, Ndola, and beyond.

## Features

- Sophisticated, modern luxury design (deep navy, gold accents, generous white space)
- Filter by category (Men / Women / Unisex), collection (Suits / Dresses / Chitenge Premium / Evening Wear), price range (ZMW)
- Elegant product cards → premium modal with large photos, fabric details, sizing
- Featured / Exclusive Collection showcase
- VIP WhatsApp inquiry buttons + floating contact button
- Client-side contact form (demo mode)
- Very easy to customize — no frameworks, no backend required
- Zambian high-end focus: premium chitenge, tailored suits, exclusive fabrics

## Technologies

- HTML5
- CSS3 (custom properties)
- Pure Vanilla JavaScript (no dependencies)

## How to Use

1. Download or clone this repository
2. Open `index.html` in any browser — works offline
3. Customize:
   - Boutique name, tagline, phone → edit `index.htm.l.`
   - Colors, fonts, luxury feel → edit  `:root` in `assets/css/style.css.`
   - Products → edit `products` array in `assets/js/script.js.`
   - Images → replace Unsplash/Pexels URLs or upload your own to `assets/images./`

### Quick Customization Examples

**Change boutique name & hero text**  
→ Open `index.html` → edit `<title>`, `<h1>`, `<p>` in hero

**Add a new luxury item**  
→ Open `assets/js/script.js` → copy-paste one object in `products = [ … ]` and edit values

**Change primary luxury color**  
→ Open `assets/css/style.css` → edit `--primary-color` (e.g., #0a2342) in `:root`
